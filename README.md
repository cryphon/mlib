# MLIB mathslibrary

This project contains headerfiles for mathematical operations


Basic Arithmetic

    Addition
    Subtraction
    Multiplication
    Division
    Modulo

Algebra

    Factorial
    Power
    Square Root
    Logarithms (natural log, log base 10, log base 2)
    Exponential Functions
    Greatest Common Divisor (GCD)
    Least Common Multiple (LCM)
    Absolute Value
